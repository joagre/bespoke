About Bespoke BBS

Offline by Design

Bespoke BBS (B3S) is a local, offline communication hub inspired by traditional Bulletin Board Systems. It runs on a small standalone appliance and includes a Forum, encrypted Direct Messaging, and simple File Sharing.

B3S appliances form their own private WiFi networks—no internet required. When two or more appliances come into range, they sync (if configured to do so) with each other automatically. This means you can post or message completely offline, and your data will eventually propagate when appliances meet. The modern internet is noisy, invasive, and bloated. B3S keeps things local, private, and simple.

The frontend runs in your browser and can be installed as a Progressive Web App (PWA) for a more native feel. Direct messages are end-to-end encrypted using keys stored securely in your browser. Even if the appliance is accessed directly, messages remain unreadable.

Note B3S is experimental, but everything you see should work. If it doesn't, assume it's a bug. Report it in the forum. Thanks.

Use it, test it, bend it a little. And please—be decent to each other.
