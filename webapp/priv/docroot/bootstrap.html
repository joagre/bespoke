Bootstrap Bespoke BBS

Provide a name for the Bespoke BBS SSID or it will default to BespokeBBS. The name should typically be descriptive for your purposes.

Note Theis bootstrap procedure will not be shown again. If you want to go through the bootstrap again, create the empty file /var/tmp/bespoke.bootstrap. If this means nothing to you, re-install the Bespoke BBS image instead.
